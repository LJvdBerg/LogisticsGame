# Game Controls

## Camera Controls
- **Pan Camera**: Left-click and drag in explore mode
- **Center on Base**: Press `Space` key
- **Toggle Build Mode**: Press `B` key

## Build Mode Controls
- **Road Tool**: Select "Road" from build bar, drag to build
- **Lumber Camp**: Select "Lumber Camp", click to place near trees
- **Refinery**: Select "Refinery", click to place near stone piles
- **Bulldoze**: Select "Bulldoze", drag to highlight, click "CONFIRM"

## Truck Management
- **Select Truck**: Press number keys `1-9` to select trucks
- **Assign Route**: Click base → select truck → click source → select resource → confirm
- **View Truck Info**: Select truck with number key, info panel appears on right

## UI Navigation
- **Build Bar**: Bottom of screen, shows available building tools
- **Resource Display**: Top-left corner, shows current resource counts
- **Truck Info Panel**: Right side, shows selected truck details
- **Bulldoze Confirm**: Red button above bulldoze tool when cells are highlighted

## Mouse Actions
- **Left Click**: Select, place buildings, interact with UI
- **Left Click + Drag**: Build roads, bulldoze areas, pan camera
- **Right Click**: Not currently used
